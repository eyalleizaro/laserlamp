# Laser Lamp Project

A custom laser/LED lamp project using an Arduino, shift register, MOSFETs, and 3D-printed parts.

## Tinkercad link:
https://www.tinkercad.com/things/chhTvBskCBa-laser-lamp?sharecode=FXg7v6mhg4UEq4_zTWdC3pU2sAC-2F1WZS-ole0Dzyg

## Features
- Controlled light patterns
- Modular electronics design
- 3D-printed enclosure
- Arduino-based control

## Main Components
- Arduino / microcontroller
- 74HC595 shift register
- MOSFETs
- Laser or LED modules
- Power supply
- 3D-printed frame/enclosure

## Safety
This project involves laser/light modules. Do not point lasers at eyes, reflective surfaces, people, animals, or vehicles. Use only safe low-power modules and proper enclosure/diffusion.
